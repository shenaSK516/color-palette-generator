# 🎨 Color Palette Generator

A modern and responsive Color Palette Generator built using HTML, CSS, and JavaScript.

## Features

* Generate random color palettes
* Copy HEX color codes with one click
* Click on a color card to copy its HEX value
* Copy success animation
* Toast notification when colors are copied
* Responsive design for desktop and mobile devices
* Modern gradient-based UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/color-palette-generator.git
```

2. Open the project folder

3. Launch `index.html` in your browser
